## Pre requisitos

<p>Download da ferramenta [Docker Desktop](https://www.docker.com/get-started/)</p>

## Manual de utilização

- Dar clone ao repositório
- Inserir este comando na consola no diretorio do repositório
```
docker compose up
```
- Abrir o client
```
http://127.0.0.1:5000/
```
